## Rules

- **No emojis as structural icons** — always use Lucide SVG icons
- **Consistent stroke width** — use the same strokeWidth across your app (default: 2)
- **Touch targets** — wrap icons in ≥44×44px interactive area (use `hitSlop` on mobile)
- **Accessibility** — always add `aria-label` to icon-only buttons
- **Dark mode** — use semantic color tokens for icon color, not hardcoded hex
