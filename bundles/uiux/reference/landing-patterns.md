# Landing Page Patterns

30 proven landing page structures organized by purpose and conversion strategy.

---

### 1. Hero + Features + CTA
- **Keywords**: hero, features, call-to-action
- **Sections**: Hero with headline/image → Value prop → Key features (3-5) → CTA section → Footer
- **CTA Placement**: Hero (sticky) + Bottom
- **Colors**: Hero: brand primary. Features: card bg #FAFAFA. CTA: contrasting accent
- **Effects**: Hero parallax, feature card hover lift, CTA glow on hover
- **Optimization**: Deep CTA placement. Contrasting color (≥7:1). Sticky navbar CTA

### 2. Hero + Testimonials + CTA
- **Keywords**: testimonials, social-proof, trust, reviews
- **Sections**: Hero → Problem statement → Solution overview → Testimonials carousel → CTA
- **CTA Placement**: Hero (sticky) + Post-testimonials
- **Colors**: Hero: brand. Testimonials: light bg #F5F5F5. Quotes: muted #666. CTA: vibrant
- **Effects**: Testimonial carousel slide, quote mark animations, avatar fade-in
- **Optimization**: Social proof before CTA. 3-5 testimonials with photo + name + role

### 3. Product Demo + Features
- **Keywords**: demo, product-demo, showcase, interactive
- **Sections**: Hero → Product video/mockup (center) → Feature breakdown → Comparison → CTA
- **CTA Placement**: Video center + CTA right/bottom
- **Colors**: Video surround: brand overlay. Feature icons: #0080FF. Text: dark #222
- **Effects**: Video play pulse, feature scroll reveals, demo interaction highlights
- **Optimization**: Embedded demo increases engagement. Interactive mockup. Auto-play muted

### 4. Minimal Single Column
- **Keywords**: minimal, simple, direct, clean
- **Sections**: Hero headline → Short description → Benefit bullets (3 max) → CTA → Footer
- **CTA Placement**: Center, large button
- **Colors**: Brand + white + accent. Buttons: ≥7:1 contrast
- **Effects**: Minimal hover. Smooth scroll. CTA scale on hover (subtle)
- **Optimization**: Single CTA focus. Large typography. Lots of whitespace. Mobile-first

### 5. Funnel (3-Step Conversion)
- **Keywords**: funnel, conversion, steps, wizard
- **Sections**: Hero → Step 1 (problem) → Step 2 (solution) → Step 3 (action) → CTA
- **CTA Placement**: Mini-CTA per step + final main CTA
- **Colors**: Steps: Red (problem) → Orange (process) → Green (solution)
- **Effects**: Step number animations, progress bar fill, smooth scroll transitions
- **Optimization**: Progressive disclosure. Essential info per step. Progress indicators

### 6. Comparison Table + CTA
- **Keywords**: comparison, table, versus
- **Sections**: Hero → Problem intro → Comparison table → Pricing → CTA
- **CTA Placement**: Right column of table + below table
- **Colors**: Alternating rows. Your product: highlighted (#FFFACD or green)
- **Effects**: Table row hover, price toggle animations, checkmark animations
- **Optimization**: Show unique value. Highlight your row. Include free trial

### 7. Lead Magnet + Form
- **Keywords**: lead, form, signup, capture, email
- **Sections**: Hero (benefit headline) → Lead magnet preview → Form (minimal) → CTA
- **CTA Placement**: Form submit button
- **Colors**: Form: clean white. Inputs: light border #CCC. CTA: brand color
- **Effects**: Form focus animations, validation animations, success confirmation
- **Optimization**: ≤3 form fields for best conversion. Valuable lead magnet preview

### 8. Pricing Page + CTA
- **Keywords**: pricing, plans, tiers, comparison
- **Sections**: Hero → Price comparison cards → Feature comparison → FAQ → Final CTA
- **CTA Placement**: Each card + sticky nav + bottom
- **Colors**: Free: grey. Starter: blue. Pro: green/gold. Enterprise: dark
- **Effects**: Price toggle (monthly/yearly), card hover lift, FAQ accordion
- **Optimization**: Recommend mid-tier ("most popular"). Annual discount 20-30%

### 9. Video-First Hero
- **Keywords**: video, hero, media, visual
- **Sections**: Hero with video background → Features overlay → Benefits → CTA
- **CTA Placement**: Overlay on video + bottom section
- **Colors**: Dark overlay 60% on video. Brand accent for CTA. White text
- **Effects**: Video autoplay muted, parallax scroll, text fade-in
- **Optimization**: 86% higher engagement with video. Add captions. Compress for perf

### 10. Scroll-Triggered Storytelling
- **Keywords**: storytelling, scroll, narrative, immersive
- **Sections**: Intro hook → Chapter 1 (problem) → Chapter 2 (journey) → Chapter 3 (solution) → CTA
- **CTA Placement**: End of each chapter (mini) + final climax
- **Colors**: Progressive reveal. Each chapter distinct color. Building intensity
- **Effects**: ScrollTrigger animations, parallax layers, progressive disclosure
- **Optimization**: 3x time-on-page. Progress indicator. Simplify animations on mobile

### 11. AI Personalization Landing
- **Keywords**: ai, personalization, dynamic, recommendation
- **Sections**: Dynamic hero (personalized) → Relevant features → Tailored testimonials → Smart CTA
- **CTA Placement**: Context-aware based on user segment
- **Effects**: Dynamic content swap, fade transitions, personalized recommendations
- **Optimization**: 20%+ conversion with personalization. Requires analytics. Fallback for new users

### 12. Waitlist / Coming Soon
- **Keywords**: waitlist, coming-soon, launch, early-access
- **Sections**: Hero with countdown → Product teaser → Email capture → Social proof (count)
- **CTA Placement**: Email form above fold + sticky on scroll
- **Colors**: Dark + accent highlights. Countdown in brand color. Urgency indicators
- **Effects**: Countdown timer, email validation, success confetti, social share
- **Optimization**: Scarcity + exclusivity. Show waitlist count. Referral program

### 13. Comparison Table Focus
- **Keywords**: comparison, versus, features
- **Sections**: Hero (problem) → Comparison matrix → Feature deep-dive → Winner CTA
- **CTA Placement**: After comparison (highlighted row) + bottom
- **Colors**: Your product highlighted (accent bg or green). Competitors neutral
- **Optimization**: 35% higher conversion. Be factual. Include pricing if favorable

### 14. Pricing-Focused Landing
- **Keywords**: pricing, cost, plans, subscription
- **Sections**: Hero (value prop) → Pricing cards (3 tiers) → Feature comparison → FAQ → CTA
- **CTA Placement**: Each pricing card + sticky nav + bottom
- **Colors**: Popular plan highlighted. Free: grey. Enterprise: dark/premium
- **Optimization**: Annual discount. Most popular badge. Address objections in FAQ

### 15. App Store Style
- **Keywords**: app, mobile, download, store
- **Sections**: Hero with device mockup → Screenshots carousel → Features → Reviews → Download
- **CTA Placement**: Download buttons (App Store + Play Store) throughout
- **Colors**: Dark/light matching app store. Star ratings gold. Device frames
- **Effects**: Device mockup rotations, screenshot slider, star animations, download pulse
- **Optimization**: Real screenshots. 4.5+ star ratings. QR code for mobile

### 16. FAQ / Documentation
- **Keywords**: faq, documentation, help, support
- **Sections**: Hero with search → Popular categories → FAQ accordion → Contact CTA
- **CTA Placement**: Search bar prominent + contact for unresolved
- **Colors**: Clean, minimal. Category icons in brand color. Success green for resolved
- **Effects**: Search autocomplete, smooth accordion, helpful feedback buttons
- **Optimization**: Reduce support tickets. Track search analytics. Related articles

### 17. Immersive / Interactive
- **Keywords**: immersive, interactive, 3D, experience
- **Sections**: Full-screen interactive → Guided product tour → Benefits revealed → CTA
- **CTA Placement**: After interaction complete + skip option
- **Colors**: Dark background for focus. Highlight interactive elements
- **Effects**: WebGL, 3D interactions, gamification, progress indicators
- **Optimization**: 40% higher engagement. Performance trade-off. Skip option. Mobile fallback

### 18. Event / Conference
- **Keywords**: event, conference, meetup, registration
- **Sections**: Hero (date/location/countdown) → Speakers → Agenda → Sponsors → Register CTA
- **CTA Placement**: Register sticky + after speakers + bottom
- **Colors**: Urgency (countdown). Event branding. Sponsor logos neutral
- **Optimization**: Early bird with deadline. Past attendee count. Multi-ticket discounts

### 19. Product Review / Ratings
- **Keywords**: reviews, ratings, social-proof, stars
- **Sections**: Hero (product + rating) → Rating breakdown → Individual reviews → Buy CTA
- **CTA Placement**: After reviews + buy alongside reviews
- **Colors**: Trust colors. Stars gold. Verified badge green
- **Optimization**: User-generated content. Verified purchases. Filter by rating

### 20. Community / Forum
- **Keywords**: community, forum, social, members
- **Sections**: Hero (community value) → Popular topics → Active members → Join CTA
- **CTA Placement**: Join prominent + after member showcase
- **Colors**: Warm, welcoming. Member photos. Topic badges in brand colors
- **Optimization**: Show active community (count, posts today). Preview content

### 21. Before-After Transformation
- **Keywords**: before-after, transformation, results
- **Sections**: Hero (problem state) → Transformation slider → How it works → Results CTA
- **CTA Placement**: After transformation reveal + bottom
- **Colors**: Muted/grey (before) vs vibrant (after). Success green for results
- **Effects**: Slider comparison, reveal animations, result counters
- **Optimization**: Visual proof. 45% higher conversion. Specific metrics. Guarantee

### 22. Marketplace / Directory
- **Keywords**: marketplace, directory, search, listing
- **Sections**: Hero (search focused) → Categories → Featured listings → Trust → CTA
- **CTA Placement**: Search bar is CTA + "List your item" in nav
- **Colors**: Search high contrast. Categories visual icons. Trust blue/green
- **Optimization**: Reduce friction to search. Popular search suggestions

### 23. Newsletter / Content First
- **Keywords**: newsletter, content, writer, blog
- **Sections**: Hero (value prop + form) → Recent issues → Social proof → About author
- **CTA Placement**: Hero inline form + sticky header form
- **Colors**: Minimalist. Paper-like background. Text focus. Accent for Subscribe
- **Optimization**: Single field (email only). "Join X,000 readers". Read sample link

### 24. Webinar Registration
- **Keywords**: webinar, training, live
- **Sections**: Hero (topic + timer + form) → What you'll learn → Speaker bio → Urgency → Form
- **CTA Placement**: Hero right side form + bottom anchor
- **Colors**: Urgency red/orange. Professional blue/navy. Form high contrast
- **Optimization**: Limited seats. "Live" indicator. Auto-fill timezone

### 25. Enterprise Gateway
- **Keywords**: enterprise, corporate, solutions, portal
- **Sections**: Hero (video/mission) → Solutions by industry → Solutions by role → Logos → Contact
- **CTA Placement**: Contact Sales (primary) + Login (secondary)
- **Colors**: Corporate navy/grey. Conservative accents
- **Optimization**: Path selection ("I am a..."). Mega menu. Trust signals prominent

### 26. Portfolio Grid
- **Keywords**: portfolio, showcase, gallery, masonry
- **Sections**: Hero (name/role) → Project grid (masonry) → About/philosophy → Contact
- **CTA Placement**: Project card hover + footer contact
- **Colors**: Neutral background (let work shine). Minimal accent
- **Optimization**: Visuals first. Filter by category. Fast loading essential

### 27. Horizontal Scroll Journey
- **Keywords**: horizontal, scroll, journey, panoramic
- **Sections**: Intro (vertical) → Journey (horizontal track) → Detail reveal → Vertical footer
- **CTA Placement**: Floating sticky CTA or end of horizontal track
- **Colors**: Continuous palette transition. Chapter colors. Progress bar
- **Effects**: Scroll-jacking (careful), parallax layers, progress indicator
- **Optimization**: Immersive product discovery. Keep navigation visible

### 28. Bento Grid Showcase
- **Keywords**: bento, grid, modular, apple-style
- **Sections**: Hero → Bento grid (key features) → Detail cards → Tech specs → CTA
- **CTA Placement**: Floating action button or bottom of grid
- **Colors**: Cards #F5F5F7 or glass. Icons vibrant brand. Text dark
- **Effects**: Hover card scale (1.02), video in cards, tilt effect, staggered reveal
- **Optimization**: Scannable value props. High density without clutter. Mobile stack

### 29. Interactive 3D Configurator
- **Keywords**: 3D, configurator, customizer, product
- **Sections**: Hero (configurator) → Feature highlight (synced) → Price/specs → Purchase
- **CTA Placement**: Inside configurator + sticky bottom bar
- **Colors**: Neutral studio background. Realistic product. Minimal UI overlay
- **Effects**: Real-time rendering, material swap, camera rotate/zoom
- **Optimization**: Increases ownership feeling. 360 view reduces returns

### 30. AI-Driven Dynamic Landing
- **Keywords**: ai, dynamic, adaptive, generative
- **Sections**: Prompt/input hero → Generated result preview → How it works → Value prop
- **CTA Placement**: Input field (hero) + "Try it" buttons
- **Colors**: Adaptive to input. Dark for compute feel. Neon accents
- **Effects**: Typing effects, shimmer loaders, morphing layouts
- **Optimization**: Immediate value demonstration. "Show, don't tell". Low friction
